# Notes Manager - Full Stack Notes Application

A simple, modern, and efficient notes taking application built with Flask backend and vanilla JavaScript frontend.

## Features

- **Add Notes**: Create new notes with title and content
- **View Notes**: Display all notes in a responsive grid layout
- **Delete Notes**: Remove notes with confirmation
- **Edit Notes**: Update existing notes (Bonus feature)
- **Character Count**: Real-time character counting for inputs
- **Timestamp**: Automatic timestamp for each note
- **Responsive Design**: Works on mobile and desktop
- **Validation**: Form validation with error messages
- **Toast Notifications**: User-friendly notifications for actions

## Tech Stack

### Frontend
- HTML5
- CSS3 (with modern flexbox/grid layouts)
- Vanilla JavaScript (ES6+)
- Font Awesome icons

### Backend
- Python Flask
- Flask-CORS for cross-origin requests

### Data Storage
- In-memory array with optional JSON file persistence
